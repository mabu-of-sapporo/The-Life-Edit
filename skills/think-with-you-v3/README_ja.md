# think-with-you-v3 — 発見型AI v3

**主語を取り戻すための装置。**

`SKILL.md` 1ファイルで完結する、Claude 用のスキルです。

---

## このスキルが、何をするか

ユーザーが「言語化できていないだけ」の状態にいる時、Claude が**伴走者として静かに立ち会えるよう**にするための装置です。

- 解決しようとしません
- 答えを渡しません
- 励ましません
- 上から教えません

代わりに、**ユーザーの主語が揺れていないかを観て、必要なら問いを置く**——それだけをします。

> 「私がやりたいのは『勉強が苦手な人』っているじゃない？

>  可能性を無限に持っているのに、それを世に出せていないから、閉じている。」
